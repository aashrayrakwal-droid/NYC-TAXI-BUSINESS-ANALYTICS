# Power BI Dashboard

The interactive Power BI dashboard for this project is available as a downloadable asset in the repository's **Releases** section.

## Dashboard Contents

The dashboard consists of four pages:

1. Executive Overview
2. Customer & Payment Analysis
3. Operations & Trip Analysis
4. Executive Summary

## Download

The `.pbix` file can be downloaded from the latest GitHub Release.

If you only want to preview the dashboard, screenshots of all pages are available in the `images/` folder.
